Privacy Policy for PostItLeaveANote

Effective Date: August 8, 2025
Last Updated: August 8, 2025

1. Data Controller

The data controller for this mobile application is Grzegorz Nowak (LeaveANote team). For inquiries related to data protection, contact: grzegorz.jan.nowak@gmail.com

2. What Data We Collect

  • Device Identifier (UID): Automatically generated to uniquely identify users.
  • Location Data (GPS): Latitude, longitude, and altitude used to position AR notes.
  • Camera Feed: Used locally to detect surfaces for AR functionality. No images or video are stored or transmitted.
  • Note Content: Includes text, skin ID, room and location identifiers, 3D position and rotation, timestamps, and author UID.
  • Technical Metadata: Includes flags like isTemporary, noteId, and timestamps (createdAt, updatedAt).

All data is stored securely on Firebase servers for synchronization and app functionality.

3. Purpose of Data Processing

  • Assign a unique identifier to each user
  • Accurately position augmented reality (AR) notes in physical space
  • Synchronize note content across devices
  • Enable camera-based AR features
  • Improve app functionality and stability

4. Legal Basis

  • User consent, provided upon first app launch
  • of the data controller, such as ensuring proper app operation and improving user experience

5. Data Sharing

  • Google Firebase, which acts as a third-party data processor for storage, synchronization, and crash reporting
  • No other third-party SDKs or advertising services are used
  • No data is shared for tracking, profiling, or advertising purposes

6. Data Retention

  • Device UID: Retained until the app is removed or the user requests deletion
  • Notes and metadata: Stored until manually deleted by the user
  • Location data: Temporarily held for positioning and synchronization only

7. User Rights

Users have the right to:

  • Access their data
  • Delete their data
  • Withdraw consent at any time
  • Restrict or object to data processing

To exercise these rights, contact: postitleaveanote@gmail.com

8. Additional Information

  • The app does not require account creation
  • No sensitive personal data (e.g. medical, religious, financial) is collected
  • We implement technical and organizational measures to safeguard user data
  • Camera input is processed locally and never stored or transmitted
  • Location data is used solely for AR positioning and is not shared or used for tracking
